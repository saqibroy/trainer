## üìö All 14 Question Types Reference

### Writing/Speaking (Open-ended)
1. **Writing**: Essay prompts with word counts
   ```
   Writing: Describe your daily routine. (min 80 words)
   ```

2. **Speaking**: Conversation prompts with time limits
   ```
   Speaking: Talk about your hobbies. (1-2 minutes)
   ```

### Reading (Comprehension)
3. **Reading**: Passage with question
   ```
   Reading: [Text: ...] | Question: What is the main idea? | Explanation
   ```

### Grammar (Exact answers)
4. **Fill-blank**: Single blank
   ```
   Ich helfe ___ Kind. | dem
   ```

5. **Multi-blank**: Multiple blanks
   ```
   Ich gebe ___ Freund ___ Buch. | dem | das
   ```

6. **Transform**: Rewrite sentence
   ```
   Transform: "Ich sehe der Mann" ‚Üí Dativ | Ich sehe dem Mann
   ```

7. **Identify**: Find mistakes
   ```
   Identify: "Ich helfe der Kind" | der Kind
   ```

8. **Error-correction**: Fix sentence
   ```
   Correct: "Ich helfe der Kind" | Ich helfe dem Kind
   ```

9. **Word-order**: Rearrange words
   ```
   Word order: gestern / ich / das Buch / gelesen / habe | Ich habe gestern das Buch gelesen
   ```

### telc B1 Specific (Exam-style)
10. **Multiple Choice**: Choose correct option
    ```
    Choice: Ich helfe ___ Kind. | A) der | B) dem | C) den | D) des | B
    ```

11. **Matching**: Pair items
    ```
    Match: helfen ‚Üí dative | geben ‚Üí dative | sehen ‚Üí accusative
    ```

12. **Sentence Building**: Arrange words
    ```
    Order: habe | Buch | das | gelesen | ich ‚Üí Ich habe das Buch gelesen
    ```

13. **Cloze Passage**: Fill paragraph
    ```
    Cloze: "Ich helfe ___ Kind. Das Kind dankt ___." | dem | mir
    ```

14. **Dialogue**: Complete conversation
    ```
    Dialogue: A: Hallo, wie geht's? | B: ___ | Mir geht's gut, danke!
    ```
