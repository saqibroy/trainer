## Tips for Conversion

1. **Keep it Simple:** Start with one exercise part at a time
2. **Test Small Batches:** Add 10-15 questions, test, then add more
3. **Use Comments:** Add `# Part X:` comments to organize (will be ignored by parser)
4. **Mix Types:** Combine different question types for variety
5. **Verify Answers:** Double-check answer keys before bulk adding
6. **Save Backups:** Keep original text file as backup
